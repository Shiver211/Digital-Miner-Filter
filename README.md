# Digital Miner Filter (数字型采矿机过滤卡)

一个 Minecraft 1.12.2 的 Forge 模组，为 Mekanism CEU 的数字采矿机添加了**过滤卡**物品，方便玩家保存和快速应用过滤器配置。

## 功能

- **过滤卡物品**：新增一个可手持的过滤卡，用于保存数字型采矿机的过滤器配置组。
- **多配置组**：一张过滤卡可以保存多个过滤器配置组，并可在它们之间切换。
- **快速应用**：手持过滤卡右键点击数字型采矿机，即可将当前选中的过滤器配置组应用到采矿机上。
- **GUI 编辑**：右键打开过滤卡 GUI 界面，方便管理配置组和过滤器。
- **JEI 支持**：GUI 中集成了 JEI 支持，可方便地从 JEI 中拖拽物品到过滤器。
- **潜行右键清空**：潜行状态下右键可清空过滤卡中的所有配置。
